## dependencies

```sh
$ gem update --system
$ gem install compass
$ npm install -g bower
$ npm install -g maildev
$ npm install -g grunt-cli
```

## run app

```sh
$ npm install
$ bower install
% grunt
$ maildev
$ DEBUG=app:* ./bin/www
```
